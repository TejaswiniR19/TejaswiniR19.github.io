- Gesture-Controlled Multi-Instrument for Motor-Impaired Users

## 📌 Introduction

Music is a powerful way to express emotions and creativity. However, traditional musical instruments require fine motor skills, making them difficult for individuals with physical disabilities.

This project introduces a gesture-controlled virtual musical instrument system that allows users to create music using simple hand gestures.

---

## 💡 Project Overview

The system uses:
- Raspberry Pi 4
- Pi Camera
- Computer Vision techniques

Hand gestures are captured and processed in real-time to control multiple musical instruments such as piano, guitar, drums, and violin.

---

## ⚙️ Working Principle

1. Camera captures hand gestures  
2. MediaPipe detects hand landmarks  
3. Gestures are recognized using computer vision  
4. Each gesture is mapped to a musical note or instrument  
5. Sound is generated using FluidSynth  

This enables users to play music without physically touching any instrument.
